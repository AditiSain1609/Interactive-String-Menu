# String Menu Project

A menu-driven Python application that allows users to perform various string operations interactively.

## Features

- Strip Spaces
- Find a Substring
- Capitalize String
- Convert to Uppercase
- Convert to Lowercase
- Convert to Title Case
- Replace a Word
- Count Occurrences
- Check Starts With
- Check Ends With
- Exit Program

## Technologies Used

- Python 3

## How to Run

1. Clone the repository

```bash
git clone https://github.com/AditiSain1609/Interactive-String-Menu
```

2. Navigate to the project folder

```bash
cd python-string-menu-project
```

3. Run the program

```bash
python string_menu.py
```
